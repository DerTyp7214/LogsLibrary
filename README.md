## To import the lib


### 1. Add it in your root build.gradle at the end of repositories
```gradle
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

### 2. Add the dependency:
```gradle
dependencies {
    implementation 'com.github.DerTyp7214:LogsLibrary:<LATEST_VERSION>'
}
```
